# 📊 SentimentScope – Real-Time Sentiment Analysis

SentimentScope is a modern, responsive web application that analyzes the emotional tone of user-provided text in real time. Using a lightweight Natural Language Processing (NLP) engine, the application classifies text into **Positive**, **Negative**, **Neutral**, or **Mixed** sentiments while processing everything locally in the browser to help preserve user privacy.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/srujanasreegangula/SentimentScope-Real-Time-Sentiment-Analysis.git
```

### Navigate to the Project

```bash
cd SentimentScope-Real-Time-Sentiment-Analysis
```

### Run the Project

Simply open **index.html** in your browser.

Or use **Live Server** in Visual Studio Code for the best experience.

---

## 💡 How It Works

1. Enter or paste any text into the input area.
2. Click **Analyze Sentiment**.
3. The application analyzes the text using an NLP-based sentiment scoring algorithm.
4. Instantly view the detected sentiment and analysis results.
